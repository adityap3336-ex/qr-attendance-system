# 📋 QR Code Based Attendance Management System

A web-based attendance management system where teachers generate QR codes and students scan them to mark attendance automatically.

🌐 **Live Demo:** [attendancemanager.great-site.net](https://attendancemanager.great-site.net)

## ✨ Features
- 🔐 Three role login system (Admin, Teacher, Student)
- 📱 QR Code generation with expiry time
- ✅ Students scan QR code to mark attendance
- 📊 Real-time attendance register with percentage
- 👨‍🏫 Teacher dashboard with session management
- 🎓 Student dashboard with attendance history
- 🔐 Admin panel to manage teachers and students
- 🚫 Duplicate attendance prevention
- ⏱ Expired QR code detection

## 🛠️ Built With
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache XAMPP
- **QR Code API:** api.qrserver.com
- **Hosting:** InfinityFree

## ⚙️ Installation
1. Clone the repository
2. Copy to XAMPP htdocs folder
3. Create database called attendance_db
4. Import SQL tables
5. Copy db_sample.php to db.php and add your credentials
6. Open localhost/attendance_system in browser
